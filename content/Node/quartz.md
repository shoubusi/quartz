# Quartz

https://quartz.jzhao.xyz/

## Get Started

```
git clone https://github.com/jackyzha0/quartz.git
cd quartz
npm i
npx quartz create
```

## Building

```
npx quartz build --serve -d ob
```

## Commit

```
npx quartz sync
```
